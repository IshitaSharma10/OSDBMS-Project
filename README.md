## Project Description

The Water Supply Complaint Management System is a database-driven project developed using MySQL.

This system is designed to manage water supply related complaints such as:

* Water leakage
* Low water pressure
* No water supply
* Dirty water
* Pipe burst

The project replaces manual complaint registration with a structured relational database system. It helps in storing, managing, updating, and tracking complaint records efficiently.

## Tables Used

## Features of the System

* Complaint Registration
* Complaint Status Tracking (Pending / Resolved / In Progress)
* Area-wise Complaint Management
* Use of Constraints (PRIMARY KEY, FOREIGN KEY, UNIQUE, NOT NULL)
* JOIN Queries for relational data retrieval
* GROUP BY Queries for report generation
* Transaction Management (START TRANSACTION, COMMIT, ROLLBACK)
